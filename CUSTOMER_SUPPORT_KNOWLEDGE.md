## 客服管道與服務時間

| 項目 | 客服使用規則 |
|---|---|
| 主要客服管道 | K大叔上網卡官方 LINE 客服（請依實際 LINE 帳號填寫） |
| 客服服務時間 | 台灣時間 `09:30-00:00` |
| 業務聯繫電話 | 請依 K大叔上網卡實際公開電話填寫；不應取代售後 LINE 客服 |
| 業務聯繫信箱 | 請依 K大叔上網卡實際公開信箱填寫；不建議作為一般售後主管道 |

客服回覆時，若涉及退換貨、無法使用、開通變更或技術排除，應請客戶優先透過 LINE 官方客服處理，以保留完整對話紀錄與圖片佐證。

## 客服受理必備資料

客戶反映上網卡或 eSIM 問題時，請先請客戶提供以下資料：

| 情境 | 需請客戶提供 |
|---|---|
| 實體 SIM 卡問題 | 訂單編號、商品名稱、目的地、抵達日期、插卡時間、手機型號、ICCID 卡號照片、錯誤畫面截圖 |
| eSIM 問題 | 訂單編號、商品名稱、目的地、掃碼或綁定時間、手機型號、完整 eSIM 信件內容、錯誤畫面截圖 |
| 無法上網 / 訊號不佳 | 所在國家城市、手機訊號截圖、行動數據與漫遊設定截圖、APN 設定截圖、目前是否可手動選網 |
| 退換貨 | 訂單編號、收到商品日期、商品是否拆封/使用、商品效期、付款方式、欲退款帳戶資訊 |

客服系統建議將「ICCID 或 eSIM 完整信件」列為必要欄位之一。原始參考資料提到，為了讓客服有效查詢問題，收到商品後應留存卡片卡號；eSIM 則應留存完整信件內容。

## 使用問題受理時限

| 規則 | 客服處理原則 |
|---|---|
| 實體卡插卡 / eSIM 綁定後 48 小時內 | 若發生無法使用，請客戶於 48 小時內透過 LINE 聯繫客服，協助線上排除 |
| 超過 48 小時才回報 | 需標示為高風險案件，K大叔上網卡可依商品與供應商規則評估是否受理 |
| 以「沒有 Wi-Fi 無法聯絡」作為退費理由 | 原始參考資料明確不接受此理由作為退費依據；K大叔上網卡客服應列為高風險退費理由 |

客服回覆時不要直接承諾退款；應先確認客戶是否在有效回報時間內、是否有提供足夠截圖與卡號資料。

## 一般使用與連線注意事項

| 問題類型 | 客服判斷與回覆重點 |
|---|---|
| 啟用與效期 | 以各商品頁的注意事項為準，不可自行推測啟用時間或效期 |
| 非覆蓋地區提前插卡 / 加入 eSIM | 應提醒客戶抵達目的地後再操作；若因提前操作造成無法使用或天數短少，官方可能不受理 |
| 網速慢 | 網速可能受電信商維修、區域、時段、裝置、3G/4G/5G 支援狀況影響；官方不接受單純因網速體感不佳申請退貨 |
| 公平使用原則 FUP | 短時間大量使用影音、遊戲、直播、系統更新或大量下載可能被限速或流量管制 |
| APP 跨國使用 | K大叔上網卡主要處理所售網卡上網功能，不保證手機內所有 APP 都能跨國正常使用 |
| 含門號商品 | 遊客短期使用性質，不應註冊為私人長期門號；電信商可能有實名抽查 |

## 熱點分享處理原則

原始參考資料對熱點分享採保守立場，K大叔上網卡客服建議沿用保守回覆：

- 分享網路時不建議超過一個裝置。
- 因手機機型差異，無法保證所有裝置都能分享熱點。
- 啟用熱點會受 FUP 公平使用原則限制。
- 若客戶重視連線品質，應建議不要分享熱點。
- 熱點分享相關問題官方可能不受理。

客服系統回覆熱點問題時，不應保證一定可用；應改說「需依商品頁與實際裝置支援為準」。

## eSIM 購買與使用規則

| 項目 | 客服處理原則 |
|---|---|
| QR Code 一次性 | eSIM QR Code 一經寄出或發送即無法收回，通常無法依消保法解約 |
| 刪除 eSIM | 客戶自行刪除 eSIM 等同自行丟棄商品，通常無法恢復或退款 |
| 提早啟用 | 若客戶因個人操作提早開啟方案，導致訊號錯亂、失效或天數短少，官方可能不負責 |
| 台灣測試 | 台灣無法收到他國基地台訊號，應抵達商品覆蓋地後再連線測試 |
| 手機支援性 | 購買前需自行確認手機是否支援 eSIM；若未確認導致不能使用，官方可能不承擔 |
| 商品頁優先 | 其餘 eSIM 規格、啟用、效期與限制以各商品頁為準 |

客服回覆 eSIM 問題時，建議先確認：

1. 手機是否支援 eSIM。
2. 是否已連上 Wi-Fi 或其他網路完成加入 eSIM。
3. 是否已抵達方案支援地區。
4. 是否有提早掃碼或提早開啟方案。
5. 是否曾刪除 eSIM。
6. 是否開啟行動數據、數據漫遊，並選到正確的 eSIM 線路。

## 開卡商品處理原則

| 項目 | 客服處理原則 |
|---|---|
| 客製開通卡 | 若商品屬於需提供開通日期的客製卡，出貨後通常不接受退換貨 |
| 抵達日期 | 訂購時需提供抵達當地日期，付款後依此安排開通 |
| 開通作業時間 | 開通作業約需 3 個工作天 |
| 開通日期變更 | 若需變更，需比原定抵達日期至少提前 3 個工作天告知；否則可能無法重新安排或退款 |
| 不可抗力 | 原廠系統維護、當地基地台損壞等因素可能造成開卡延遲 |
| 商品頁優先 | 每張卡的開卡細節仍以商品頁說明為準 |

客服建立工單時，若商品屬於開卡商品，應將「抵達當地日期」與「原訂開通日期」列為必填資訊。

## 實體 SIM 卡障礙排除

客戶反映實體卡無法使用時，客服可依序請客戶確認：

1. 是否已抵達商品覆蓋地。
2. 是否依說明書完成設定。
3. 行動數據是否開啟。
4. 數據漫遊是否開啟。
5. APN 是否依商品說明設定正確。
6. 手機內是否有殘留描述檔需刪除。
7. 電信商是否需手動選擇。
8. 雙卡手機是否將 SIM 放在卡槽 1，並清空卡槽 2，避免互相干擾。

若仍無法使用，請客戶在插卡後 48 小時內透過 LINE 聯繫，並提供 ICCID、手機型號、設定截圖與所在城市。

## 手機與裝置相容性

| 項目 | 客服提醒 |
|---|---|
| 裝置支援性 | 市售手機型號與販售渠道複雜，客服不能保證所有手機都支援 |
| eSIM 支援 | eSIM 是否可用取決於手機是否支援 eSIM 功能，與一般 SIM 卡支援性不同 |
| 實體卡建議 | iPhone 8 以後機型通常可使用實體 SIM 卡 |
| 不確定時 | 請客戶洽詢手機品牌商或購買通路確認規格 |
| 特定日本商品 | 日本電信商關閉部分網路頻段，可能影響部分手機使用，需查看商品頁說明 |

客服不要自行判斷陌生型號是否可用；應請客戶提供手機完整型號，必要時請客戶向品牌商確認。

## 退換貨處理重點

| 項目 | 客服處理原則 |
|---|---|
| 可退貨範圍 | 官網商品排除 eSIM、須開通商品、即期網卡、機場取貨商品後，符合條件者才可能享 7 天鑑賞期 |
| 退貨提出時間 | 自收到商品隔日起算 7 日內，需透過 LINE 向客服提出 |
| 商品狀態 | 包裝完整、未拆封、未使用，且效期需大於提出退貨日 60 天 |
| 退回方式 | 退貨提出日起 7 日內用郵局掛號寄回，包裹內需附訂購資訊與銀行帳號 |
| 退款金額 | 會扣除系統交易手續費與原寄送運費；客戶寄回運費由客戶負擔 |
| 退款時間 | 官方收到並確認商品狀態後，約 30 天內完成退款 |
| 換貨 | 除須開通與即期網卡外，若有換貨需求需直接聯繫 LINE 客服，依客服提供方式處理 |

退換貨類客服回覆應避免直接使用「可以退」或「不能退」的絕對句，除非已確認商品類型、狀態、日期與後台啟用狀態。

## 防詐騙提醒

原始參考資料頁尾提醒，公司不會透過電話要求顧客操作網路銀行或 ATM。K大叔上網卡客服也應採用相同的防詐原則。

客服遇到客戶詢問可疑電話、付款異常或自稱客服來電時，應提醒：

- 不要提供個人資料、信用卡資料、驗證碼或銀行帳戶資訊。
- 不要依陌生來電指示操作 ATM 或網路銀行。
- 可撥打 165 反詐騙專線，並透過官方 LINE 客服確認。

## 客服回覆範本

### 無法上網

您好，為了協助您排除連線問題，請先確認已抵達商品支援地區，並開啟行動數據與數據漫遊。請再依商品說明確認 APN、描述檔與電信商選擇是否正確。  
若仍無法使用，請提供訂單編號、商品名稱、手機型號、所在城市、ICCID 或 eSIM 完整信件內容，以及目前設定畫面截圖，我們會協助您確認。

### eSIM 掃碼或啟用問題

您好，eSIM 加入手機時需要連上 Wi-Fi 或其他網路，且請確認手機支援 eSIM。若您已掃碼，請勿自行刪除 eSIM，避免商品無法恢復。  
請提供訂單編號、手機型號、掃碼時間、是否已抵達目的地，以及錯誤畫面截圖，我們會協助確認。

### 熱點分享

您好，熱點分享需依商品頁與手機裝置實際支援為準，且分享時可能受到公平使用原則影響。若希望維持較穩定的連線品質，建議不要分享熱點，或不要同時分享超過一個裝置。

### 退換貨

您好，退換貨需先確認商品類型、是否拆封或使用、收到商品日期與商品效期。eSIM、須開通商品、即期網卡與機場取貨商品可能有不同限制。  
請提供訂單編號、商品照片、收到日期與欲處理原因，我們會依商品狀態與官方規範協助確認。
