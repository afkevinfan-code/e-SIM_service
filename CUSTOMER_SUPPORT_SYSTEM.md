# K大叔上網卡客服系統

## 啟動方式

```bash
./run_support_system.sh
```

啟動後打開：

```text
http://127.0.0.1:8765
```

如需換 port：

```bash
./run_support_system.sh --port 8777
```

## 主要功能

- 依客戶問題自動判斷常見分類，例如 eSIM 安裝、無法上網、APN、熱點分享、效期啟用、退款更換。
- 產生可複製給客戶的初步客服回覆。
- 依目的地、天數、流量與商品類型查詢 `card.xlsx` / `eSim.xlsx` 候選方案。
- 建立工單並寫入 `support_tickets.jsonl`，方便日後追蹤。

## 客服知識文件

- 官方購物前須知、退換貨與使用問題相關內容已整理於 `CUSTOMER_SUPPORT_KNOWLEDGE.md`。
- 客服回覆正式送出前，請先對照該文件與商品頁資訊，特別是 48 小時內回報、eSIM 一次性、開卡商品、熱點分享、退換貨條件等高風險規則。

## 建議客服流程

1. 將客戶名稱、訂單編號或聯絡資訊填入「客戶名稱或訂單編號」。
2. 在「目的地 / 商品關鍵字」輸入目的地與條件，例如 `日本 5天 eSIM`。
3. 貼上客戶原始問題。
4. 點「產生客服建議」取得回覆草稿與商品候選方案。
5. 若需要追蹤，點「建立工單」留下紀錄。

## 注意事項

- 商品候選方案沿用 `quote_lookup.py` 的查詢與排序邏輯。
- `報價成本` 不會自動視為客戶最終售價，對外報價前仍需確認售價策略。
- 若商品資料未明確寫出熱點、效期、APN 或啟用規則，客服回覆應標示待確認，不要自行補規格。
- 工單紀錄檔 `support_tickets.jsonl` 可能包含客戶資訊，對外分享專案前請先檢查內容。
